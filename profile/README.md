## Overview

Trickl is an open source self-hosted metrics observability pipeline with built in cardinality control for small-to-medium sized development teams. It deploys a full backend that receives metrics from your instrumented apps, applies intelligent cardinality control recommendations, and surfaces everything through a Grafana front end, all on your own infrastructure.

Trickl automates the provisioning and configuration of a self-managed metrics backedn on AWS, simplifying setup and reducing operational burden.

Read our case study for more information:
https://trickl-app.github.io/

Please use our CLI tool for an easy deployment:
https://github.com/Trickl-app/trickl-cli
